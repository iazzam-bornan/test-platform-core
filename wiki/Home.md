# @testplatform/core Wiki

Welcome to the `@testplatform/core` wiki! This is the complete knowledge base for the Docker-based test environment orchestrator.

## What is @testplatform/core?

A TypeScript library that lets you define test environments declaratively and orchestrate them with Docker Compose. You describe your services, infrastructure, and tests — the platform handles the rest: spinning up containers, waiting for healthchecks, running tests, streaming results, and cleaning up.
